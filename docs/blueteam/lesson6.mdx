## Introduction

Preventing attacks is just as important as detecting them.

One of the main ways Blue Teams reduce security risks is through system hardening.

System hardening means reducing vulnerabilities and strengthening system security before attackers can exploit weaknesses.

## What System Hardening Means

System hardening is the process of making systems more secure by reducing potential attack points.

This is done by removing unnecessary features, improving configurations, and ensuring systems are properly protected.

The goal is to make systems more difficult for attackers to exploit.

## Software Updates and Patching

Many cyber attacks take advantage of known software vulnerabilities.

Organizations must regularly update systems to fix security flaws.

These updates are called patches.

Applying patches helps close security gaps before attackers can exploit them.

## Limiting Permissions

Users and systems should only have the permissions they actually need.

This concept is often called least privilege.

Examples include:
- employees only accessing systems related to their job
- restricting administrative privileges
- limiting access to sensitive files

Limiting permissions reduces the impact of compromised accounts.

## Disabling Unnecessary Services

Systems sometimes run services that are not needed.

Each additional service creates a potential entry point for attackers.

By disabling unnecessary services, organizations reduce their attack surface.

## Secure Configurations

Systems must be configured securely.

Examples of secure configuration practices include:
- strong authentication policies
- proper access controls
- secure network settings
- monitoring enabled

These settings strengthen overall system protection.

## Key Takeaways

- System hardening strengthens systems before attacks occur.
- Software updates fix security vulnerabilities.
- Limiting permissions reduces potential damage.
- Disabling unnecessary services reduces attack opportunities.
- Secure configurations improve overall system protection.
