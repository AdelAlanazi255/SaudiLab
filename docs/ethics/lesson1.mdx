## 1. Cyber Security Is Not Just About Technology

Cyber security is often connected to tools, software, and technical skills.

But cyber security also includes decisions about:
- what is right
- what is allowed
- what is responsible

A person can have technical skills and still use them in the wrong way.

---

## 2. Why Ethics Matter

Ethics helps people decide what they should and should not do.

In cyber security, ethics matters because professionals may handle:
- private data
- secure systems
- sensitive information
- powerful security tools

Without ethics, these tools and skills could be misused.

---

## 3. How Actions Can Affect Real People

Cyber security decisions affect real people, not just computers.

Examples:
- exposing private information can harm users
- misusing access can damage trust
- testing systems without permission can create legal and personal problems

Actions in digital spaces can have real-world consequences.

---

## 4. Cyber Security Without Ethics Can Be Harmful

Technical ability alone is not enough.

Examples of harmful behavior:
- reading data without permission
- testing websites you do not own
- sharing vulnerabilities irresponsibly
- misusing access to systems

Even if a person is skilled, their actions can still be unethical or illegal.

---

## 5. Good Security Starts with Responsibility

Responsible cyber security means:
- respecting people and systems
- following rules and laws
- using skills to protect, not harm
- thinking about consequences before acting

This course will help you understand those boundaries.

---

## 6. Key Takeaways

- Cyber security is not only technical; it also involves ethics.
- Ethical decisions help protect people, systems, and data.
- Technical skill without responsibility can be harmful.
- Cyber security professionals must think about consequences and boundaries.
