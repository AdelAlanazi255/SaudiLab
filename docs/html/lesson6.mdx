# Lesson 6: Lists

Lists help you group related items in a clean way. In this lesson, you will learn how to build numbered and bullet lists in HTML.

## What You'll Learn
- What <code>&lt;ul&gt;</code> and <code>&lt;li&gt;</code> are
- What <code>&lt;ol&gt;</code> and <code>&lt;li&gt;</code> are
- The difference between ordered and unordered lists
- How to create a simple nested list

---

## Ordered vs Unordered Lists

Use <code>&lt;ul&gt;</code> when item order does not matter (bullet points).

Use <code>&lt;ol&gt;</code> when order matters (steps or ranking).

Each list item is written with <code>&lt;li&gt;</code>.

---

## Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Lists</title>
  </head>
  <body>
    <h1>Lists in HTML</h1>

    <h2>My Favorite Snacks</h2>
    <ul>
      <li>Popcorn</li>
      <li>Dates</li>
      <li>Yogurt</li>
    </ul>

    <h2>Steps to Make Tea</h2>
    <ol>
      <li>Boil water</li>
      <li>Add a tea bag to a cup</li>
      <li>Pour hot water into the cup</li>
      <li>Wait 2 minutes and serve</li>
    </ol>

    <h2>Weekend Plan</h2>
    <ul>
      <li>
        Study
        <ul>
          <li>Review HTML lessons</li>
          <li>Practice in the Try page</li>
        </ul>
      </li>
      <li>Exercise</li>
    </ul>
  </body>
</html>
```

## Explanation

<div className="explanationBlock">
  <ul className="explanationText">
    <li><code>&lt;ul&gt;</code> - creates an unordered (bulleted) list</li>
    <li><code>&lt;ol&gt;</code> - creates an ordered (numbered) list</li>
    <li><code>&lt;li&gt;</code> - creates one list item</li>
    <li>A nested list means placing a list inside another <code>&lt;li&gt;</code></li>
  </ul>
</div>
