# Lesson 10: Semantic Layout

## Semantic structure
```html
<header>
  <h1>My Website</h1>
</header>

<nav>
  <a href="#">Home</a>
</nav>

<main>
  <section>
    <h2>About</h2>
    <p>Welcome to my site.</p>
  </section>
</main>

<footer>
  <p>Copyright 2026</p>
</footer>
```

## Why semantic HTML matters
- Better SEO
- Accessibility
- Cleaner code
