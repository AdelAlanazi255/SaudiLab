# Lesson 4: Links

Links connect pages and help users navigate between information. In this lesson, you will learn how to build useful links for websites, sections, and email.

## What You'll Learn
- Basic link structure with <code>&lt;a&gt;</code>
- <code>href</code> vs link text
- Open links in a new tab with <code>target="_blank"</code>

---

## Core Concepts

- A basic link is written as <code>&lt;a href="..."&gt;Link text&lt;/a&gt;</code>.
- <code>href</code> is the destination, while the visible words are the link text.
- Absolute URL: full web address like <code>https://example.com</code>.
- Relative URL: path inside your site like <code>/some-page</code>.
- To open a link in a new tab, use <code>target="_blank"</code>.

---

## Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Links</title>
  </head>
  <body>
    <h1>My Favorite Links</h1>

    <p>
      Visit <a href="https://www.google.com" target="_blank">Google</a>
      to search the web.
    </p>

    <p>
      Watch videos on
      <a href="https://www.youtube.com" target="_blank">YouTube</a>.
    </p>

    <p>
      Learn something new on
      <a href="https://en.wikipedia.org" target="_blank">Wikipedia</a>.
    </p>

    <p>
      Explore developer documentation on
      <a href="https://developer.mozilla.org" target="_blank">MDN Web Docs</a>.
    </p>
  </body>
</html>
```

## Explanation

<div className="explanationBlock">
  <ul className="explanationText">
    <li><code>&lt;a&gt;</code> - creates a clickable link</li>
    <li><code>href</code> - sets the destination URL</li>
    <li><code>target</code> - controls where the link opens (such as a new tab)</li>
  </ul>
</div>

---

## Common Mistakes

- Missing <code>https://</code> in external links
- Using vague text like "click here" instead of descriptive link text
