# Lesson 7: Tables

Tables are used to show tabular data in rows and columns. They are great for schedules, plans, and simple comparisons.

## Table Basics

Use a table when information naturally fits into rows and columns.

A header row usually uses <code>&lt;th&gt;</code>, and normal data rows use <code>&lt;td&gt;</code>.

You can also add <code>&lt;caption&gt;</code> to describe what the table is about.

---

## Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Tables</title>
  </head>
  <body>
    <h1>Weekly Study Plan</h1>

    <table border="1" style="border-collapse: collapse;">
      <caption>My Weekly Study Plan</caption>
      <tr>
        <th>Day</th>
        <th>Subject</th>
        <th>Duration</th>
      </tr>
      <tr>
        <td>Sunday</td>
        <td>HTML</td>
        <td>45 min</td>
      </tr>
      <tr>
        <td>Monday</td>
        <td>CSS</td>
        <td>45 min</td>
      </tr>
      <tr>
        <td>Tuesday</td>
        <td>JavaScript</td>
        <td>60 min</td>
      </tr>
    </table>
  </body>
</html>
```

## Explanation

<div className="explanationBlock">
  <ul className="explanationText">
    <li><code>&lt;table&gt;</code> - wraps the whole table</li>
    <li><code>&lt;tr&gt;</code> - creates one row</li>
    <li><code>&lt;th&gt;</code> - creates a header cell</li>
    <li><code>&lt;td&gt;</code> - creates a data cell</li>
  </ul>
</div>
