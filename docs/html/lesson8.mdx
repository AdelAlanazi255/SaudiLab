# Lesson 8: Forms Basics

Forms are used to collect user input such as names, emails, and passwords. In this lesson, you will build a simple sign up form.

## What You'll Learn
- What a form is and why we use it
- How to use <code>&lt;form&gt;</code>, <code>&lt;label&gt;</code>, <code>&lt;input&gt;</code>, and <code>&lt;button&gt;</code>
- Basic input types: <code>text</code>, <code>email</code>, <code>password</code>
- How <code>placeholder</code> and <code>required</code> help users

---

## Simple Form Structure

Each input should have a matching label so users know what to enter.

Use the same value for <code>label for</code> and input <code>id</code> to connect them correctly.

---

## Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Forms Basics</title>
  </head>
  <body>
    <h1>Sign Up</h1>

    <form onsubmit="return false;">
      <label for="name">Name</label><br>
      <input id="name" type="text" placeholder="Enter your name" required><br><br>

      <label for="email">Email</label><br>
      <input id="email" type="email" placeholder="Enter your email" required><br><br>

      <label for="password">Password</label><br>
      <input id="password" type="password" placeholder="Create a password" required><br><br>

      <button type="submit">Create Account</button>
    </form>
  </body>
</html>
```

## Explanation

<div className="explanationBlock">
  <ul className="explanationText">
    <li><code>&lt;form&gt;</code> - wraps all input fields</li>
    <li><code>&lt;label&gt;</code> - describes the field and links with matching <code>for</code>/<code>id</code></li>
    <li><code>&lt;input&gt;</code> - where users type data</li>
    <li><code>&lt;button&gt;</code> - submits the form</li>
  </ul>
</div>
