What is HTML?

HTML stands for **HyperText Markup Language**.  
It is the foundation of every website.

HTML uses elements (tags) to structure content like:

- Headings
- Paragraphs
- Links
- Images

---

Your First HTML Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>
```
