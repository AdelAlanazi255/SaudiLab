<div className="sectionTitle">What is HTML?</div>

HTML stands for HyperText Markup Language.

It is the language used to structure content on web pages. With HTML you can create headings, paragraphs, links, images and many other elements that make up a webpage.

---

<div className="sectionTitle">Your First HTML Example</div>

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>
```

<div className="explanationBlock">
  <div className="sectionTitle">Explanation</div>
  <div className="explanationText">This simple HTML document contains:</div>

  <ul className="explanationText">
    <li><code>&lt;!DOCTYPE html&gt;</code> - tells the browser this is an HTML5 document</li>
    <li><code>&lt;head&gt;</code> - contains information about the page such as the title</li>
    <li><code>&lt;body&gt;</code> - contains the visible content displayed on the webpage</li>
    <li><code>&lt;p&gt;</code> - defines a paragraph of text on the webpage</li>
  </ul>
</div>
