# Lesson 9: Form Inputs

Form inputs let users type and choose information. In this lesson, you will learn common input types and useful attributes.

---

<div className="sectionTitle">Common Input Types</div>

Use <code>text</code> for names, <code>email</code> for email addresses, and <code>password</code> for passwords.

Use <code>number</code> when users enter numbers, and <code>radio</code>/<code>checkbox</code> when users choose options.

---

<div className="sectionTitle">Helpful Attributes</div>

<code>placeholder</code> shows a hint inside the input.

<code>required</code> makes a field mandatory.

For numeric inputs, <code>min</code> and <code>max</code> set allowed limits.

<code>name</code> and <code>value</code> help identify selected values in forms.

---

<div className="sectionTitle">Example: Simple Form</div>

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Form Inputs</title>
  </head>
  <body>
    <h1>Quick Form</h1>

    <form onsubmit="return false;">
      <label for="name">Name</label>
      <input id="name" type="text" placeholder="Enter your name" />

      <button type="submit">Submit</button>
    </form>
  </body>
</html>
```
