# Lesson 3: Text Formatting

## Bold and Strong

```html
<p>This is <b>bold</b> text.</p>
<p>This is <strong>important</strong> text.</p>
```

---

## Italic and Emphasis

```html
<p>This is <i>italic</i> text.</p>
<p>This is <em>emphasized</em> text.</p>
```

---

## Line Break

```html
<p>Line one<br>Line two</p>
```

---

## Horizontal Line

```html
<hr>
```

---

## Full Example

```html
<p>This is <strong>important</strong> text.</p>
<hr>
<p>This is <em>emphasized</em> text.</p>
```
