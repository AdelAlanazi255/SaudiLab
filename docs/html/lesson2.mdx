# Lesson 2: Headings and Paragraphs

Headings and paragraphs are the most common building blocks on a page. Headings create structure, and paragraphs hold readable text.

## What You'll Learn
- What headings are (h1-h6)
- When to use paragraphs (<code>&lt;p&gt;</code>)
- How to structure a simple page with headings + text

---

## Headings

- <code>&lt;h1&gt;</code> is the main page topic (usually one per page).
- <code>&lt;h2&gt;</code> and <code>&lt;h3&gt;</code> are used for sub-sections.
- Don&apos;t choose heading level based on size; choose based on structure and importance.

---

## Paragraphs

- Use <code>&lt;p&gt;</code> for normal readable text.
- Browsers ignore extra spaces and line breaks in HTML, so paragraphs help with spacing and readability.

---

## Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Headings and Paragraphs</title>
  </head>
  <body>
    <h1>My First Blog Post</h1>
    <p>This is my first paragraph on the web.</p>

    <h2>What I learned today</h2>
    <p>Headings create structure and paragraphs hold readable text.</p>

    <h3>Next step</h3>
    <p>I will practice writing clean HTML.</p>
  </body>
</html>
```
