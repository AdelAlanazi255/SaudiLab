## 1. What is Network Scanning

Network scanning is the process of discovering devices and services connected to a network.

Security professionals use scanning to understand what systems exist and how they communicate.

Scanning helps answer questions like:
- what devices are online
- what services are running
- which ports are open

---

## 2. Why Network Scanning is Important

Understanding the network is a key step in security testing.

Organizations need to know:
- which systems are exposed to the internet
- which services are running
- whether unnecessary services are open

If unnecessary services are exposed, the risk of attack can increase.

---

## 3. Understanding Ports and Services

Ports are communication endpoints used by software services.

Examples of common services include:
- web services
- email services
- remote access services

Each service listens on a specific port.

If a port is open, a service is available to communicate.

---

## 4. What Nmap Does

Nmap is a well-known network scanning tool used by security professionals.

It helps identify:
- active devices on a network
- open ports
- running services
- potential network exposure

Nmap helps teams build a clearer view of network structure.

---

## 5. Why Security Professionals Use Network Scanning

Ethical security professionals use network scanning to identify possible risks.

Network scanning helps teams:
- discover unknown systems
- identify exposed services
- verify network configurations
- improve security defenses

Scanning systems without permission can be illegal.

Authorization is always required before testing.

---

## 6. Key Takeaways

- Network scanning helps discover systems and services on a network.
- Ports are communication points used by network services.
- Open ports indicate that services are available.
- Tools like Nmap help security professionals understand network exposure.
- Network scanning must always be performed with authorization.
