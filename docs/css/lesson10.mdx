## What You Will Learn
- How to combine your CSS skills in one small layout
- How to style a simple heading, content card, and button
- How colors and spacing work together

## Introduction
This lesson brings together the CSS skills you have already learned.

You do not need advanced CSS to build something clean and useful.

## Build a Small Page Section
Think of a simple mini page with:
- a title
- a short paragraph
- a small info card
- a button

## Style the Main Container
Start by styling the page background and a main container.

```css
body {
  background-color: #eef2ff;
  font-family: Arial, sans-serif;
}

.layout {
  max-width: 520px;
  margin: 24px auto;
  padding: 20px;
}
```

## Style the Text
Use simple text styling to make the heading and paragraph clear.

```css
h1 {
  color: #111827;
}

p {
  color: #334155;
}
```

## Style a Content Card
Use box model skills for a clean content section.

```css
.card {
  background-color: #ffffff;
  border: 2px solid #2563eb;
  border-radius: 10px;
  padding: 16px;
  margin-top: 12px;
}
```

## Style a Button
Add a simple button and hover style.

```css
button {
  background-color: #2563eb;
  color: #ffffff;
  border: none;
  border-radius: 8px;
  padding: 10px 16px;
}

button:hover {
  background-color: #1d4ed8;
}
```

## Keep Spacing Consistent
Use margin and padding values that feel balanced.

Small spacing changes can make your layout look much better.

## Quick Recap
- You combined selectors, colors, spacing, and buttons
- You built a mini layout with simple CSS rules
- This is a strong beginner project using skills you already know
