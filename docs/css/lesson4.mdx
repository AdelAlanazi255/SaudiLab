## Introduction
In earlier lessons, you used selectors like `h1` and `p`.

That works well, but sometimes you want more control.

## Why We Need More Control
If you style all `p` elements, every paragraph gets the same style.

Sometimes you only want to style:
- a few selected elements
- one special element

That is where **classes** and **IDs** help.

## What Is a Class?
A class is a reusable label you add to elements in HTML.

You write class selectors in CSS with a dot `.`.

```css
.highlight {
  color: #2563eb;
}
```

If multiple elements use `class="highlight"`, they all get this style.

## What Is an ID?
An ID is usually for one specific element.

You write ID selectors in CSS with a hash `#`.

```css
#special-note {
  background-color: #facc15;
  color: #111827;
}
```

This targets the element with `id="special-note"`.

## HTML + CSS Example
```html
<p class="highlight">This text uses a class.</p>
<p id="special-note">This text uses an ID.</p>
```

```css
.highlight {
  color: #2563eb;
}

#special-note {
  background-color: #facc15;
  color: #111827;
}
```

## Class vs ID (Simple Rule)
- Use a **class** when you want to reuse the same style on multiple elements.
- Use an **ID** when you want to style one special element.

## Quick Recap
- `.class-name` targets elements by class
- `#id-name` targets one specific element by id
- Classes are reusable
- IDs are usually unique
