## Introduction to Layout
Sometimes you want boxes to line up in a clean row or column.

Flexbox is a simple layout tool that helps with this.

## What Is Flexbox?
Flexbox arranges elements inside a container.

When a container uses `display: flex`, its child elements become flexible items that can be arranged easily.

## Flex Container
```css
.container {
  display: flex;
}
```

After this, items inside the container sit next to each other in a row.

## Flex Direction
`flex-direction` controls item direction.

```css
.container {
  display: flex;
  flex-direction: row;
}
```

- `row` = horizontal
- `column` = vertical

## Simple Alignment
`justify-content` controls horizontal placement.

`align-items` controls vertical alignment.

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

## Example Layout
```css
.container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}
```

## Quick Recap
- Flexbox arranges items inside a container
- `display: flex` turns a container into a flex container
- `flex-direction` changes row or column direction
- `justify-content` and `align-items` help with simple alignment
