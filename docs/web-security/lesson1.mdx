## 1. What is Web Security

Web security means protecting websites, web applications, and online services from attacks and misuse.

Most modern services run on websites:
- social media
- online stores
- banking
- education platforms
- government services

Because so much activity happens online, protecting websites is extremely important.

---

## 2. Why Websites Need Protection

Websites store and process important data.

Examples include:
- usernames and passwords
- personal information
- payment details
- messages and private data

If a website is not secure, attackers may access or steal this information.

Web security helps prevent unauthorized access and protects users.

---

## 3. Common Things Websites Protect

Web security focuses on protecting several important areas:

User accounts  
Sensitive data  
Website functionality  
Server resources

If these are compromised, websites may stop working or users may lose trust.

---

## 4. Who Tries to Attack Websites

Attackers may target websites for different reasons.

Examples include:
- financial gain
- stealing personal information
- spreading malware
- damaging a company's reputation

Some attacks are automated, meaning bots scan the internet looking for vulnerable websites.

---

## 5. Why Even Small Websites Are Targeted

Many beginners think attackers only target large companies.

In reality, many attacks are automated.

Attackers use tools that scan thousands of websites for vulnerabilities.

Small websites can be easier targets because they often have weaker security.

---

## 6. Key Takeaways

- Web security focuses on protecting websites and online services.
- Websites store important data such as user accounts and personal information.
- Attackers target websites for financial gain, data theft, or disruption.
- Automated attacks constantly search for vulnerable websites.
- Even small websites need proper security protections.
