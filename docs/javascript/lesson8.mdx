## 1. What an Object Is

An object stores multiple related values together.

Example idea:
A person has a name, age, and job.

Instead of separate variables:

~~~javascript
let name = "Ali";
let age = 22;
let job = "Student";
~~~

We can use one object:

~~~javascript
let person = {
  name: "Ali",
  age: 22,
  job: "Student"
};
~~~

Objects store information using **key: value pairs**.

---

## 2. Object Properties

Each piece of data inside an object is called a property.

Example properties:

~~~javascript
person.name
person.age
~~~

---

## 3. Accessing Object Values

Use a dot `.` to access properties.

~~~javascript
let person = {
  name: "Ali",
  age: 22,
  job: "Student"
};

console.log(person.name);
console.log(person.age);
~~~

---

## 4. Updating Object Values

You can change values inside an object.

~~~javascript
person.age = 23;

console.log(person);
~~~

---

## 5. Quick Recap

- Objects group related data.
- They use key:value pairs.
- We access properties using dot notation.
