What You'll Learn
- Create objects with key-value pairs
- Read and update properties

---

Core Concept

Objects group related data using named properties. They are useful for representing entities like users or products.

---

Example

~~~javascript
const user = {
  name: "Sara",
  level: "Beginner"
};

console.log(user.name);
~~~
