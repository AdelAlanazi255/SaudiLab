## 1. Introduction

A variable is a named place to store a value.

You can think of it like a labeled box:
- the label is the variable name
- the value is what is inside the box

---

## 2. Why Variables Matter

Variables help you store information and use it again later.

This makes your code easier to read and update.

---

## 3. Creating a Variable with `let`

Use `let` to create a variable:

~~~javascript
let name = "Adel";
~~~

Here, `name` stores text.

---

## 4. Simple Data Types

In this lesson, we use 3 basic data types:

- **string** = text  
Example: `"Adel"`

- **number** = numeric value  
Example: `21`

- **boolean** = `true` or `false`  
Example: `true`

---

## 5. Showing Variable Values

Use `console.log()` to print variable values in the Console Output panel.

~~~javascript
let name = "Adel";
let age = 21;
let isStudent = true;

console.log(name);
console.log(age);
console.log(isStudent);
~~~

Press **RUN** to see the output.

---

## 6. Quick Recap

- A variable stores a value.
- `let` creates a variable.
- Strings are text, numbers are numbers, and booleans are true/false.
- `console.log()` prints values to the Console Output panel.
