What You'll Learn
- Declare variables with let and const
- Work with strings, numbers, and booleans
- Inspect values with console.log

---

Core Concept

Variables store data you want to reuse. Use const for values that should not be reassigned, and let when values can change.

---

Example

~~~javascript
const studentName = "Ahmad";
let score = 10;
score = score + 5;
console.log(studentName, score);
~~~

Practice
