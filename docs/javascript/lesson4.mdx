What You'll Learn
- Write if and else statements
- Check multiple conditions with else if
- Run code only when a condition is true

---

Core Concept

Conditionals control program flow. They allow your code to choose different paths based on data.

---

Example

~~~javascript
const temperature = 32;

if (temperature > 30) {
  console.log("It is hot today.");
} else {
  console.log("Weather is mild.");
}
~~~

Practice
