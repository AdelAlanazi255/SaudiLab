## 1. Introduction

Programs often need to make decisions.

A conditional lets JavaScript choose what to do based on whether something is true or false.

---

## 2. Using `if`

`if` checks a condition.

If the condition is true, that block runs.

~~~javascript
let age = 20;

if (age >= 18) {
  console.log("Adult");
}
~~~

---

## 3. Using `else`

`else` runs when the `if` condition is false.

~~~javascript
let age = 16;

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
~~~

---

## 4. Using `else if`

`else if` gives one extra choice in the middle.

~~~javascript
let score = 75;

if (score >= 90) {
  console.log("Excellent");
} else if (score >= 60) {
  console.log("Good job");
} else {
  console.log("Keep practicing");
}
~~~

---

## 5. Simple Real-World Example

Imagine checking test results:
- high score = "Excellent"
- middle score = "Good job"
- lower score = "Keep practicing"

Conditionals help JavaScript choose the correct message.

---

## 6. Quick Recap

- Conditionals help JavaScript make decisions.
- `if` checks a condition.
- `else` runs when the condition is false.
- `else if` adds another possible branch.
