What You'll Learn
- Repeat code with for loops
- Use while loops for condition-based repetition
- Avoid infinite loops by updating counters

---

Core Concept

Loops run a block of code multiple times. They help when processing lists or repeating tasks.

---

Example

~~~javascript
for (let i = 1; i <= 3; i += 1) {
  console.log("Step", i);
}
~~~
