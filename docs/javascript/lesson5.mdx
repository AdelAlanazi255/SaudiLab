## 1. Why Loops Exist

Sometimes we want to repeat the same action many times.

For example, printing numbers from 1 to 5.

Without a loop:

~~~javascript
console.log(1);
console.log(2);
console.log(3);
console.log(4);
console.log(5);
~~~

Loops help us avoid repeating code.

---

## 2. The `for` Loop

Basic structure:

~~~javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
~~~

Simple meaning:
- `let i = 1` = start at 1
- `i <= 5` = keep running while this is true
- `i++` = increase by 1 each time

---

## 3. Example: Counting Numbers

~~~javascript
for (let i = 1; i <= 10; i++) {
  console.log(i);
}
~~~

This loop runs 10 times.

---

## 4. Example: Repeating a Message

~~~javascript
for (let i = 1; i <= 3; i++) {
  console.log("Practice makes progress!");
}
~~~

---

## 5. Quick Recap

- Loops repeat actions.
- `for` loops are useful for counting and repeating tasks.
