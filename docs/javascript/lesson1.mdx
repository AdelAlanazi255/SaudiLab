## 1. Introduction

JavaScript is the language that adds behavior to websites.

Think of it like this:
- HTML = structure
- CSS = style
- JavaScript = behavior

HTML builds the page, CSS makes it look good, and JavaScript makes it do things.

---

## 2. What JavaScript Can Do

JavaScript can:
- calculate values
- control webpage behavior
- respond to user actions

Example:
- A button click shows a message
- A total price is calculated
- A page updates without reloading

---

## 3. Your First JavaScript Code

```javascript
console.log("Hello SaudiLab!");
```

`console.log()` prints text to the **Console Output** panel.

Press **RUN** to see your message.

---

## 4. Editing the Code

You can change the text inside the quotes.

```javascript
console.log("Learning JavaScript!");
```

Press **RUN** again and the Console Output will show the new text.

---

## 5. Quick Recap

JavaScript:
- runs logic
- produces output
- can control web pages
