## 1. What the DOM Is

DOM means **Document Object Model**.

It is the structure of the webpage that JavaScript can interact with.

Simple idea:
- HTML builds the page
- CSS styles the page
- JavaScript can change the page

JavaScript can:
- change text
- update elements
- react to user actions

---

## 2. Selecting an Element

Use:

~~~javascript
document.querySelector("h1")
~~~

This tells JavaScript to find the first `<h1>` element on the page.

---

## 3. Changing Text Content

~~~javascript
document.querySelector("h1").textContent = "Hello JavaScript!";
~~~

JavaScript finds the element and replaces its text.

---

## 4. Console + DOM Example

~~~javascript
console.log("Updating the page...");

document.querySelector("h1").textContent = "Welcome to SaudiLab!";
~~~

In this lesson:
- Console Output shows script messages
- Preview shows webpage changes

---

## 5. Quick Recap

- The DOM represents the webpage.
- JavaScript can change HTML content.
- `querySelector()` helps us find elements.
