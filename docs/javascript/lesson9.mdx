What You'll Learn
- Select elements from the page

---

Core Concept

The DOM represents your page as objects. JavaScript can read and modify those objects to update the UI.

---

Example

~~~javascript
document.body.innerHTML = "<h1 id=\"title\">Hello</h1>";
const title = document.getElementById("title");
title.textContent = "Hello from DOM";
~~~

Practice

Create an element, set its text, and append it to the page.
