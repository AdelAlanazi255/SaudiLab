## 1. What a Function Is

A function is a reusable block of code.

Instead of writing the same code many times, we can place it inside a function and run it whenever we need.

---

## 2. Creating a Function

Basic syntax:

~~~javascript
function greet() {
  console.log("Hello!");
}
~~~

Simple meaning:
- `function` starts a function
- `greet` is the function name
- code inside `{ }` runs when we call the function

---

## 3. Calling a Function

Functions do nothing until we call them.

~~~javascript
function greet() {
  console.log("Hello!");
}

greet();
~~~

---

## 4. Functions with Simple Parameters

~~~javascript
function greet(name) {
  console.log("Hello " + name);
}

greet("Adel");
greet("Student");
~~~

The value inside the parentheses becomes the variable inside the function.

---

## 5. Simple Real-World Example

~~~javascript
function showScore(score) {
  console.log("Your score is " + score);
}

showScore(85);
~~~

---

## 6. Quick Recap

- Functions store reusable code.
- You create them with `function`.
- You run them by calling their name.
