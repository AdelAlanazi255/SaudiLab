What You'll Learn
- Define reusable functions
- Pass parameters to functions

---

Core Concept

Functions group logic into reusable blocks. They improve readability and reduce repeated code.

---

Example

~~~javascript
function greet(name) {
  return `Welcome, ${name}!`;
}

console.log(greet("Student"));
~~~

Practice
