## 1. What an Array Is

Sometimes we want to store multiple values in one place.

Instead of:

~~~javascript
let student1 = "Ali";
let student2 = "Sara";
let student3 = "Omar";
~~~

We can use one array:

~~~javascript
let students = ["Ali", "Sara", "Omar"];
~~~

Arrays store a list of values.

---

## 2. Array Index

Each item in an array has a position called an index.

Important rule:
Arrays start counting from **0**, not 1.

~~~javascript
let students = ["Ali", "Sara", "Omar"];

// students[0] is "Ali"
// students[1] is "Sara"
// students[2] is "Omar"
~~~

---

## 3. Accessing Array Values

Use square brackets `[ ]` to access items.

~~~javascript
let fruits = ["Apple", "Banana", "Orange"];

console.log(fruits[0]);
console.log(fruits[1]);
console.log(fruits[2]);
~~~

---

## 4. Changing an Array Value

You can update a value by using its index.

~~~javascript
let fruits = ["Apple", "Banana", "Orange"];

fruits[1] = "Mango";

console.log(fruits);
~~~

---

## 5. Quick Recap

- Arrays store multiple values.
- We access items using indexes.
- Arrays start counting from 0.
