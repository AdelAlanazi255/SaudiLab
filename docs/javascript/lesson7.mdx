What You'll Learn
- Create and read arrays
- Access items by index
- Loop through array values

---

Core Concept

Arrays store ordered collections of values. Each item has an index starting at 0.

---

Example

~~~javascript
const topics = ["HTML", "CSS", "JavaScript"];
console.log(topics[0]);
console.log(topics.length);
~~~

---

Practice

Add a new item to the array and print all items with a loop.
