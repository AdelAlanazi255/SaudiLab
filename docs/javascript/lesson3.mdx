What You'll Learn
- Use arithmetic operators
- Compare values with comparison operators
- Build expressions that return true or false

---

Core Concept

Operators let you calculate and compare values. Expressions combine values and operators to produce a new result.

---

Example

~~~javascript
const a = 8;
const b = 3;
console.log(a + b);
console.log(a > b);
~~~

---

Practice

Try changing the values and test +, -, *, /, and comparison operators.
