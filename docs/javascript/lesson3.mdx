## 1. What an Operator Is

An operator is a symbol that performs an action on values.

Example:

~~~javascript
5 + 3
~~~

Here, `+` is the operator.

---

## 2. Arithmetic Operators

In this lesson, we use:
- `+` add
- `-` subtract
- `*` multiply
- `/` divide

Simple example:

~~~javascript
let result = 5 + 3;
console.log(result);
~~~

---

## 3. Expressions

When we combine values and operators, we create an expression.

Example:

~~~javascript
let total = 10 + 5;
~~~

`10 + 5` is an expression.

---

## 4. Comparison Operators (Simple)

We also use operators to compare values:
- `>`
- `<`
- `===`

Example:

~~~javascript
let age = 20;

console.log(age > 18);
console.log(age === 20);
~~~

These comparisons return `true` or `false`.

---

## 5. Beginner Recap

- Operators perform actions.
- Arithmetic operators do basic math.
- Comparison operators check values and return `true` or `false`.
