## 1. What is Malware

Malware stands for **malicious software**.

It is software created to damage systems, steal information, or give attackers control of devices.

Malware can affect:
- computers
- smartphones
- servers
- networks

Attackers often use malware to gain unauthorized access or collect sensitive information.

---

## 2. Common Types of Malware

Common malware types include:

- Virus: malware that attaches to files and spreads when those files are opened.
- Trojan: malware disguised as normal software to trick users into installing it.
- Spyware: software that secretly monitors user activity and collects information.
- Adware: software that shows unwanted advertisements.

Different malware types have different goals, but all create security risks.

---

## 3. What is Ransomware

Ransomware is a type of malware that locks or encrypts files on a device.

Attackers then demand payment (a ransom) to restore access.

Example:
A victim cannot open important files, and a message appears demanding payment for a decryption key.

Ransomware can affect individuals, small businesses, and large organizations.

---

## 4. How Malware Spreads

Malware often spreads through:

- malicious email attachments
- fake software downloads
- infected websites
- pirated software or cracked applications
- malicious links in messages

Attackers rely on users opening files or links without realizing they are unsafe.

---

## 5. Protecting Yourself from Malware

Use these simple prevention habits:

- only download software from trusted sources
- avoid opening suspicious email attachments
- keep operating systems and applications updated
- use antivirus or security software when possible
- avoid clicking unknown links

Awareness and safe daily habits can greatly reduce infection risk.

---

## 6. Key Takeaways

- Malware is malicious software designed to harm systems or steal information.
- There are different malware types such as viruses, trojans, and spyware.
- Ransomware locks files and demands payment to restore access.
- Malware often spreads through attachments, downloads, and malicious links.
- Safe browsing habits help prevent malware infections.
