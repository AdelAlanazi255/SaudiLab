Account protection basics

- **Passwords:** weak and reused passwords are easier to guess or steal.
- **Multi-Factor Authentication:** adds a second verification step, so a password alone is not enough for attackers.
- **Phishing:** fake urgent messages try to make you click unsafe links and reveal credentials.
- **Updates:** security updates patch known weaknesses before attackers exploit them.
