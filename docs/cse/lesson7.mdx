## 1. What is a Network

A network is a group of devices connected together to share data or access the internet.

Examples of devices connected to networks:
- computers
- smartphones
- tablets
- smart TVs
- servers

These devices communicate through routers and internet connections.

---

## 2. Why Network Security Matters

Attackers often target networks to gain access to systems or data.

If a network is not secure, attackers may be able to:
- intercept communications
- access connected devices
- steal information
- spread malware across devices

Protecting networks helps prevent unauthorized access.

---

## 3. Protecting Home and Public Networks

Use these simple practices:

- secure your home Wi-Fi with a strong password
- avoid connecting to unknown or suspicious public Wi-Fi networks
- verify the correct network name before connecting
- avoid accessing sensitive accounts (such as banking) on public Wi-Fi when possible

Attackers sometimes create fake Wi-Fi networks to trick users.

---

## 4. Device Security Basics

Individual devices also need protection.

Examples of device security practices:
- use screen locks on phones and computers
- install applications only from trusted sources
- avoid connecting unknown USB devices
- use built-in security tools when available

If a device is compromised, attackers may use it to access other systems on the network.

---

## 5. Software Updates and Security

Software updates are important for security.

Updates often include **security patches** that fix vulnerabilities.

If systems are not updated, attackers may exploit known weaknesses.

Safe update habits:
- enable automatic updates
- regularly update operating systems and apps

---

## 6. Key Takeaways

- Networks connect devices and allow them to communicate.
- Attackers may target insecure networks to access systems.
- Strong Wi-Fi passwords help protect home networks.
- Public Wi-Fi networks should be used carefully.
- Keeping devices and software updated improves security.
